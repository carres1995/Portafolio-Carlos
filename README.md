###### Responsivo #########

###### MOODLE BUENAS PRACTICAS ######

    1. Minificacion del CSS para reducir tiempos de carga del archivo poniendo todas las etiquetas lineales
    2. uso eficiente de selectores utilizando las especificaciones solo necesarias del codigo.
    3. clases reutilizables segun identificacion de etiquetas con misma caracteristica.
    4. prelacion en los archivos por practicidad en este caso en un archivo base.css para que los cambios se ejecuten en orden critico.
